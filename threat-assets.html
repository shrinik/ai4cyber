Critical Asset Identification

The following IT assets have been deemed to be critical for the hospital industry

Following five critical assets were identified

  1. IT Infrastructure - This includes hardware like servers and software like operating systems
  2. Information Systems - - Set of software applications and data required for the administrative functioning of the hospital
  3. Patient Data - - This includes patients PHI, PII and other types of medical records
  4. IoT Medical Devices - - These are connected devices and systems used for patient monitoring and diagnostics, etc.
  5. Mobile Devices - - These are used for patient registration, case charts, etc.
